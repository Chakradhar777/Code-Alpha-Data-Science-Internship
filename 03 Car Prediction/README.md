# Car Price Prediction using Machine Learning

## Overview

This project predicts the selling price of used cars using Machine Learning.

## Features

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Data Visualization
- Random Forest Regression
- Model Evaluation
- Feature Importance Analysis

## Dataset Features

- Car Name
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner
- Car Age

## Visualizations

1. Selling Price Distribution
2. Correlation Heatmap
3. Fuel Type Distribution
4. Transmission Distribution
5. Car Age vs Selling Price
6. Present Price vs Selling Price
7. Actual vs Predicted Prices
8. Feature Importance

## Evaluation Metrics

- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

## Run

```bash
pip install -r requirements.txt
python car_price_prediction.py
```
