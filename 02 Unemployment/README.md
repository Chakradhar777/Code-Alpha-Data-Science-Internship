# Unemployment Analysis with Python

## Project Objective

Analyze unemployment trends in India using Python and visualize important patterns in the data.

## Features

- Data Cleaning
- Exploratory Data Analysis
- State-wise Unemployment Analysis
- Monthly Trend Analysis
- Rural vs Urban Comparison
- Correlation Analysis
- Covid-19 Impact Analysis
- Data Visualization

## Visualizations

1. Unemployment Rate Distribution
2. Top 10 States by Unemployment
3. Monthly Trend Analysis
4. Rural vs Urban Comparison
5. Labour Participation Analysis
6. Correlation Heatmap
7. Covid-19 Impact Trend
8. Regional Comparison

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Run

```bash
pip install -r requirements.txt
python unemployment_analysis.py
```
